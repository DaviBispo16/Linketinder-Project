# Projeto Groovy (Cadastro de Empresas / Candidatos)

Projeto simples em **Groovy** para cadastro e listagem via terminal (entrada com `Scanner`).

---

## Requisitos

- **Java (JDK) 11+**
- Uma das opções abaixo:
  - **Groovy 3+** (para executar como script)
  - **Gradle Wrapper** (`./gradlew`) para executar como projeto Gradle

---

## Como executar

### Opção A — Executar como script Groovy

1. Verifique se o Java está instalado:
```bash
java -version
```

2. Verifique se o Groovy está instalado:
```bash
groovy -version
```
3. Execute o arquivo principal:
```bash
groovy Main.groovy
```
