# Linketinder - Frontend

Projeto desenvolvido com HTML, CSS e TypeScript para conectar candidatos e empresas.

## Como Executar o Projeto

Para rodar o projeto localmente, siga os passos abaixo:

### 1. Compilar o TypeScript
O projeto utiliza TypeScript, que precisa ser transpilado para JavaScript para que o navegador possa executá-lo. Use o comando abaixo na pasta raiz do frontend:

```bash
npx tsc
```
Isso gerará os arquivos compilados na pasta `dist/`.


### 2. Remover a pasta dist
```bash
rm -rf dist
```
